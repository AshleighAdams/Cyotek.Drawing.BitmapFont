Based on code from the article AngelCode bitmap font parsing using C# <https://www.cyotek.com/blog/angelcode-bitmap-font-parsing-using-csharp>.

Some code in this demonstration comes from assorted blog posts on cyotek.com, see links in individual code files for details.

JSON support courtesy of SimpleJSON <https://github.com/Bunny83/SimpleJSON>.
